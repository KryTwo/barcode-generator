требования:
0. Диалоговое окно со всеми перечисленными ниже функциями. Библиотека Fyne.
1. Базово принимает файл csv с двумя колонками: Штрихкод (ШК) и название товара.
2. Генерирует картинку с наложением названия товара на ШК. ШК формата code 128.
3. Можно выбирать расположение названия товара (слева, справа, сверху, снизу), 
центровать, изменять размер, шрифт, цвет текста и фон.
3.1 (рассмотреть необходимость встраивания шрифтов в бинарник)
4. Можно изменять размер ШК, в пикселях, мм, см.
5. При экспорте генерирует изображение формата А4 в портретном виде,
    с максимальным заполнением для экономии бумаги для печати. 
    (пример, если 22 ШК на выходе, а в лист влезает две колонки по 10 ШК, 
    то выдает 2 картинки по 20 + 2 ШК.
6. Возможность выбора типа разделителя.
7. (расширенный функционал) Возможность обработки большого csv файла с автоматическим
    выбором нужных колонок.
8. Возможность сохранить значения (размеры и все остальные параметры) как пресет. 
    При перезапуске программы, открывается последний закрытый пресет, остается возможность 
    выбора других пресетов. Где храним? Вероятно json
9. Формирование log файла для записи ошибок (для отправки разработчику)
10. Тестирование. Юнит тесты
11. На случай изменения исходного файла в доп настройках реализовать возможность добавлять 
    ключевые слова, определяющие принадлежность столбов. Например:
    сейчас Штрихкод называется "id", при переименовании (например в code128) его в дальнейшем,
    чтобы каждый раз не исправлять файл, можно добавить соответствие id - code128
    Тем самым обеспечавается продолжительность жизни приложения.
